Problem: Complement of Base 10 Integer

Approach:
We are given a non-negative integer n, and we need to find its bitwise complement. The complement is obtained by flipping every bit in the binary representation of the number: 0 becomes 1, and 1 becomes 0.

One important edge case is when n = 0. The binary representation of 0 is "0", and its complement should be "1", so we return 1.

For other numbers, we process the bits one by one. We use a variable i that represents the current bit position (starting from the least significant bit). In each iteration, we flip the corresponding bit of n using the XOR (^) operation.

The XOR operation with 1 flips the bit at that position. After processing one bit, we shift i left (i << 1) to move to the next bit.

We continue this process until i becomes greater than n, meaning all relevant bits have been processed. The resulting number is the bitwise complement.

Time Complexity: O(log n)
Space Complexity: O(1)


```cpp

class Solution {
public:
    int bitwiseComplement(int n) {

        if (n == 0) return 1;

        int i = 1;
        while (i <= n)
        {
            n = i^n;
            i = i << 1;
        }

        return n;
        
    }
};
