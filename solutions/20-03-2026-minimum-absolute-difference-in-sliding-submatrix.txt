Problem: Minimum Absolute Difference in sliding Submatrix

Approach:
We are given a grid and a size k. For every k × k submatrix, we need to find the minimum absolute difference between any two distinct elements.

We slide a k × k window over the grid. For each starting position (i, j):

Collect all elements inside the k × k submatrix into a list.
Sort the list.
Check the difference between adjacent elements in the sorted list.
The minimum difference will always be between neighbors after sorting.
Ignore equal elements (since we want distinct values).
If all elements are the same, return 0.

We store this value for each submatrix in the answer matrix.

Time Complexity:
O((m × n) × k² log k)
Space Complexity: O(k²)


```cpp

class Solution {
public:
    vector<vector<int>> minAbsDiff(vector<vector<int>>& grid, int k) {
        int m = grid.size(), n = grid[0].size();
        vector<vector<int>> ans(m - k + 1, vector<int>(n - k + 1, 0));

        for (int i = 0; i <= m - k; i++) {
            for (int j = 0; j <= n - k; j++) {
                
                vector<int> vals;

                // collect elements of k x k submatrix
                for (int x = i; x < i + k; x++) {
                    for (int y = j; y < j + k; y++) {
                        vals.push_back(grid[x][y]);
                    }
                }

                // sort values
                sort(vals.begin(), vals.end());

                int minDiff = INT_MAX;

                // compute minimum absolute difference
                for (int t = 1; t < vals.size(); t++) {
                    if (vals[t] != vals[t - 1]) {
                        minDiff = min(minDiff, vals[t] - vals[t - 1]);
                    }
                }

                // if all elements are same
                if (minDiff == INT_MAX) minDiff = 0;

                ans[i][j] = minDiff;
            }
        }

        return ans;
    }
};
